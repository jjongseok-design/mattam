

# 식당 위치 데이터 정확도 보정 계획

## 현재 상황
- 데이터베이스에 44개+ 식당이 등록되어 있으며, 일부(예: 미소 차이니즈레스토랑)의 좌표가 실제 위치와 1km 이상 차이남
- 좌표가 부정확하면 지도 마커가 엉뚱한 곳에 표시됨

## 진행 방식: 1차 선별 수정 → 2차 전체 검증

### 1차: 우선 보정 대상 선별 및 수정
- 웹 검색으로 각 식당의 실제 주소/좌표를 확인
- 오차가 큰 식당부터 데이터베이스의 `lat`, `lng`, `address` 값을 UPDATE
- 미소 차이니즈레스토랑 포함, 주소가 불명확하거나 좌표가 의심스러운 식당 우선 처리

### 2차: 나머지 전체 식당 검증
- 1차에서 다루지 못한 식당들도 순차적으로 검증 및 보정
- 필요시 여러 번에 나눠서 진행

### 기술적 접근
- `supabase insert tool`(UPDATE 쿼리)로 좌표/주소 데이터 직접 수정
- 코드 변경 없음 -- 데이터만 보정하면 지도 마커가 자동으로 올바른 위치에 표시됨

### 제한 사항
- 폐업했거나 이전한 식당은 검색 결과가 없을 수 있음 → 발견 시 사용자에게 알림
- 한 번에 전체를 완벽하게 하기 어려울 수 있어 여러 라운드로 나눠 진행
